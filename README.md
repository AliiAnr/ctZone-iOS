## Technologies Used

- Swift & SwiftUI
- UserDefaults & Core Data
- MVVM 
- SOLID Principle

## Requirements

- iOS 16.0+
- Xcode 10.0

## Installation
1. Clone Repository
```
https://github.com/AliiAnr/ctZone-iOS.git 
```
2. Open in Xcode
- Ensure you are using the latest version of Xcode.
- Open the .xcodeproj or .xcworkspace file.
3. Run the App
